stop_forum_spam by ldenman

Api wrapper for querying a spammer based on ip, email, or username information stored on http://stopforumspam.com

Dependencies

jnunemaker's httparty

Install

sudo gem install gemcutter sudo gem tumble sudo gem install httpary sudo gem install stop_forum_spam
